# NOTICE — Binary/executable payloads removed

This skill is part of the **Security Suite** distribution. The following upstream
payload artifacts have been **deliberately excluded** to avoid shipping executable
or AV-triggering content: the EICAR test file, Flash `.swf` exploit, and the
`directory-traversal.zip` / `max-length.zip` archives.

The benign filename-injection test payloads (command-substitution, null-byte) and the
README references remain. For the full upstream payload set, see:

- SecLists Payloads: https://github.com/danielmiessler/SecLists/tree/master/Payloads

Use only against systems you own or are explicitly authorized to test.
