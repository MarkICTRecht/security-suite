# NOTICE — Live web-shell samples removed

This skill is part of the **Security Suite** distribution. To keep the plugin safe
to install and store, the **live, runnable web-shell source files** that ship in the
upstream SecLists / awesome-skills-security repository (PHP/ASP/ASPX/JSP/CFM/WAR/EXE
backdoors, reverse shells, the `laudanum` and `Vtiger` shell kits, `nc.exe`, etc.)
have been **deliberately excluded**.

What remains here is the **instructional `SKILL.md`** plus benign metadata
(`backdoor_list.txt` filename list, licence/credits/README files). Use this skill for
detection signatures, naming patterns, and defensive analysis.

If you are an authorized tester who needs the actual shell samples (e.g. to test a
WAF or an EDR detection), pull them directly from the upstream source:

- SecLists Web-Shells: https://github.com/danielmiessler/SecLists/tree/master/Web-Shells

Only use such samples in an isolated lab against systems you own or are explicitly
authorized to test.
